問題
---------

$N \times N$ 正方行列 $M$ が与えられます。行列式をmod 998244353で求めてください。

制約
---------

- $1 \leq N \leq 500$
- $0 \leq m_{ij} < 998244353$

入力
---------

```
$N$
$M_{11}$ $M_{12}$ ... $M_{1N}$
$M_{21}$ $M_{22}$ ... $M_{2N}$
:
$M_{N1}$ $M_{N2}$ ... $M_{NN}$
```
