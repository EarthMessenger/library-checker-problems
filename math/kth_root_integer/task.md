## {{keyword.statement}}
この問題は $T$ ケース与えられます。

$A, K$ が与えられます。$\textrm{floor}(a^{1/k})$ を求めてください。

## {{keyword.constraints}}

- $1 \leq T \leq 500,000$
- $0 \leq A < 2^{64}$
- $1 \leq K \leq 64$

## {{keyword.input}}

```
$T$
$A_0$ $K_0$
$A_1$ $K_1$
:
$A_{T-1}$ $K_{T-1}$
```

## {{keyword.output}}

各行に、求めたものを出力してください。
