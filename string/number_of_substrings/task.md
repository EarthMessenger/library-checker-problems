## @{keyword.statement}

@{lang.en}

Given a length $N$ string $S$. Count the number of __distinct__ substrings of $S$.

@{lang.ja}

長さ $N$ の文字列 $S$ が与えられます。$S$ の相異なる部分文字列の個数を求めてください。

@{lang.end}

## @{keyword.constraints}

@{lang.en}

- $1 \leq N \leq 500,000$
- Each character of $S$ is lowercase English letters.

@{lang.ja}

- $1 \leq N \leq 500,000$
- $S$ は英小文字からなる。

@{lang.end}

## @{keyword.input}

~~~
$S$
~~~

## @{keyword.output}

~~~
$ans$
~~~
