Problem Statement / 問題文
---------

長さ $N$ の文字列 $S$ が与えられます。$S$ の相違なる部分文字列の個数を求めてください。

Constraints / 制約
---------

- $1 \leq N \leq 500,000$
- $S$ は英小文字のみからなる

Input / 入力
---------

~~~
$S$
~~~

Output / 出力
---------

~~~
$ans$
~~~
