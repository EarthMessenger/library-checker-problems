## {{keyword.statement}}
長さ $N$ の文字列 $S$ が与えられます。$S$ の suffix array $a_i$ を求めてください。


## {{keyword.constraints}}

- $1 \leq N \leq 500,000$
- $S$ は英小文字からなる


## {{keyword.input}}

```
$S$
```

## {{keyword.output}}

```
$a_0$ $a_1$ ... $a_{N-2}$ $a_{N-1}$
```
