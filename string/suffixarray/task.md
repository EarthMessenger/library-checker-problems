Problem Statement (Japanese) / 問題文 (日本語)
---------

長さ $N$ の文字列 $S$ が与えられます。$S$ の suffix array $a_i$ を求めてください。


Constraints / 制約
---------

- $1 \leq N \leq 500,000$
- $S$ は英小文字からなる


Input / 入力
---------

```
$S$
```

Output / 出力
---------

```
$a_0$ $a_1$ ... $a_{N-2}$ $a_{N-1}$
```
