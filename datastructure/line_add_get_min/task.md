問題文
---------

$N$ 本の直線 $y = a_i x + b_i$ が存在します。 $Q$ 個のクエリを処理してください。

- `0 $a$ $b$`: 直線 $y = ax + b$ を追加
- `1 $p$`: $x = p$ での $y$ 座標の最小を求める。

制約
---------

- $1 \leq N, Q \leq 200,000$
- $|a_i|, |p| \leq 10^9$
- $|b_i| \leq 10^{18}$

入力
---------

~~~
$N$ $Q$
$a_0$ $b_0$
$a_1$ $b_1$
:
$a_{N-1}$ $b_{N-1}$
$\textrm{Query}_0$
$\textrm{Query}_1$
:
$\textrm{Query}_{Q - 1}$
~~~
